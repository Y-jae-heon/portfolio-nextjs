# PD Report

## 비주얼 방향

- 이력서형 템플릿이 아니라 차분한 제품 소개 페이지 톤으로 설계했다.
- 따뜻한 오프화이트 배경, 얇은 그리드감, 코발트 포인트 컬러로 고급스럽지만 과장되지 않은 인상을 목표로 했다.

## 레이아웃 원칙

- 히어로에서 성과 카드와 CTA를 함께 보여 신뢰 신호를 먼저 준다.
- 프로젝트는 카드 요약과 상세 케이스 스터디 블록으로 나눠 스캔성과 깊이를 동시에 확보한다.
- 역량, 경력, 문제 해결 방식은 키워드 나열이 아니라 판단 기준과 실행력 중심으로 정리한다.

## UI 품질 기준

- 강한 위계, 짧은 문단, 일관된 카드 시스템
- 모바일과 데스크톱 모두에서 무너지지 않는 간격 체계
- 장식적 모션 배제, 의미 있는 hover와 focus만 사용
- progress bar, icon grid, resume template 패턴 금지

## 구현 반영 포인트

- shadcn/ui 스타일의 `Button`, `Card`, `Badge`, `Separator` 프리미티브 패턴 반영
- 한국어 카피가 길어져도 읽기 부담이 커지지 않도록 좁은 텍스트 폭과 넉넉한 줄간격 적용
