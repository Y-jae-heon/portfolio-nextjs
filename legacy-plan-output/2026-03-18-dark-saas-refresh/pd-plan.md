# PD Plan

## 시각 방향

- 딥 네이비 기반의 절제된 SaaS / dashboard UI
- 검정이 아니라 푸른 기운이 도는 어두운 배경
- 블루는 CTA, 링크, 메트릭, 약한 배경 글로우에만 사용

## 타이포 규칙

- 제목/본문: Pretendard
- 숫자/eyebrow/날짜/태그: JetBrains Mono
- eyebrow tracking은 `0.08em ~ 0.14em`
- Hero 헤드라인 tracking은 `-0.03em` 내외
- 섹션 헤딩 tracking은 `-0.02em` 내외
- 한국어 본문 line-height는 `1.7+`

## 컴포넌트 방향

- Button: primary는 블루 채움, secondary는 dark surface + border
- Card: surface 대비와 border 중심, 그림자는 깊이 보조 수준
- Badge: developer metadata처럼 작고 정돈된 pill
- Timeline: 밝은 선보다 절제된 라인과 포인트 dot

## 피해야 할 것

- 강한 네온
- 과한 blur/glass
- 작은 본문에 넓은 자간
- 모든 카드가 같은 시각 밀도를 가지는 상태
