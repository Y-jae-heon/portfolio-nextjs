# Style Refresh Plan Set

## 목적

- 기존 문서를 덮어쓰지 않고, 2026-03-18 기준 스타일 리프레시 전용 plan/handoff 세트를 분리한다.
- 멀티 에이전트가 동일한 기준으로 움직일 수 있도록 색상, 폰트, UX 우선순위, 구현 순서를 고정한다.

## 고정 방향

- Primary: `#000000`
- Secondary: `#FFFFFF`
- Accent: `#6B7280`
- Optional Accent: `#EF4444`
- Heading / Body: `Pretendard`
- Code / Emphasis / Numbers: `JetBrains Mono`

## 의도

- 애플 스타일에 가까운 절제된 신뢰감
- 장식보다 콘텐츠 우선
- "실력으로 승부"하는 엔지니어 포트폴리오 인상

## 실행 순서

1. PM plan 확정
2. PM handoff를 기준으로 PD/FE 병렬 착수
3. PD 기준 반영 후 FE 구현
4. FE handoff를 기준으로 QA 검수
