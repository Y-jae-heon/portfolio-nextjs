# FE Plan

## 목표

- Next.js + TypeScript + Tailwind 기반의 정적 우선 포트폴리오를 구현한다.
- 실제 페이지 텍스트는 한국어 중심으로 작성하고, 한국 회사 리뷰 기준에 맞는 문장 톤을 유지한다. 필요한 직무명과 기술 용어는 영어를 제한적으로 사용한다.
- UI 프레임워크는 `shadcn/ui` 기준으로 사용하되, 정적 우선 구조를 해치지 않는 범위에서 필요한 컴포넌트만 적용한다.

## 수행 범위

- 로컬 타입드 콘텐츠 구조 설계
- 단일 페이지 섹션 구현
- 모바일 내비게이션 구현
- 접근성 기본값 적용
- 메타데이터와 정적 배포 적합성 정리
- `Button`, `Card`, `Badge`, `Separator`, `Accordion` 계열 컴포넌트 구조 반영

## 산출물

- 구현 코드
- 한국어 카피가 반영된 페이지
- 빌드 및 린트 가능한 상태

## 검수 기준

- 서버 의존성이 없어야 한다.
- 모든 주요 섹션이 존재해야 한다.
- 프로젝트 3건이 구조화되어 있어야 한다.
- 첫 화면에서 직무 포지셔닝과 성과 신호가 보여야 한다.
- 텍스트가 한국 채용 담당자 기준으로 자연스러워야 한다.
