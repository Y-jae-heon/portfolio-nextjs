# QA Plan: Theme Refresh

## 목표

- 새 색상/폰트 시스템이 신뢰도와 가독성을 실제로 높였는지 검증한다.
- 리브랜딩 후에도 정적 배포, 반응형, 접근성, 기본 인터랙션 품질이 유지되는지 확인한다.

## 수행 범위

- 폰트 로딩 성공 여부 확인
- 헤드라인/본문/mono 역할 분리 검증
- Hero, Projects, Summary, Contact 우선 점검
- 모바일/데스크톱 스크린샷 비교
- CTA, nav, focus ring, contrast 회귀 확인

## 핵심 검수 기준

- 첫 화면이 더 절제되고 더 신뢰감 있게 보여야 한다.
- 회색 보조 텍스트가 흐리거나 대비 부족으로 보이지 않아야 한다.
- 붉은 강조는 소수 지점에만 쓰여야 하며, 반복 노출되지 않아야 한다.
- mono 폰트가 과용되지 않아야 한다.
- 기존보다 카드/배경 장식이 줄었는데도 빈약해 보이지 않아야 한다.

## 산출물

- 리브랜딩 전후 시각 차이 메모
- Playwright 스크린샷
- contrast 및 focus 상태 점검 결과
- 최종 release recommendation
