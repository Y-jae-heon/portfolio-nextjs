# Handoff: PD to FE

## 시각 시스템 전달

- Display 폰트 제거
- Pretendard를 기본 활자축으로 사용
- JetBrains Mono는 지표, 라벨, 메타 정보, 강조 숫자에만 사용
- 배경은 거의 흰색 평면으로 두고, 필요 시 매우 약한 질감만 허용

## 컴포넌트 지침

- Button: 채움/보더 대비를 명확히 하되 과한 radius를 피한다.
- Card: border 중심, 얕은 shadow 또는 무그림자 우선
- Badge: 정보 태그처럼 작고 단정하게
- Section header: eyebrow는 Mono, title은 Pretendard, description은 Gray

## 레이아웃 지침

- Hero는 한 번에 읽히는 구조를 유지한다.
- Project 카드는 카드 자체보다 텍스트 블록 구조가 먼저 보이게 한다.
- Metric 블록은 숫자와 설명의 대비로 리듬을 만든다.

## 접근성 지침

- hover보다 focus를 더 분명하게
- Gray 본문은 대비 부족이 생기지 않게 조정
- 빨강은 상태성보다 강조성에 한정
