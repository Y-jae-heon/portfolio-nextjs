# Handoff: FE to QA

## QA가 확인할 핵심

- 모든 섹션이 새 팔레트와 폰트 체계를 일관되게 따르는지
- 블루 토큰이나 기존 display 폰트 흔적이 남아 있지 않은지
- Hero, Projects, Capability, Timeline에서 텍스트 위계가 명확한지
- 모바일에서 Mono 메타 정보가 과밀하지 않은지

## 회귀 위험

- sticky header 대비 저하
- 카드와 배지의 강조 과다 또는 대비 부족
- focus ring 약화
- 빨강 포인트의 과용

## 검증 결과물

- 데스크톱/모바일 주요 화면 캡처
- 발견 이슈 목록
- 배포 전 수정 권고
