# Handoff: FE to QA

## 변경 영역

- 전역 컬러 토큰과 배경
- 버튼/카드/배지/구분선 프리미티브
- Hero / Projects / Timeline / Contact 섹션 위계
- letter-spacing 및 line-height 조정

## 집중 검증 포인트

- eyebrow와 badge의 자간이 자연스러운가
- 다크 배경에서 보조 텍스트가 충분히 읽히는가
- Hero 성과 패널이 메인 메시지를 방해하지 않는가
- Project 카드와 상세 블록의 정보 우선순위가 분명한가
- 모바일 sticky header와 메뉴가 안정적인가

## 회귀 위험

- 배경 효과 과다
- 카드 대비 부족
- focus-visible 약화
- 메타 텍스트의 과소 강조
