# SKILL: nextjs-static-build

## Use this skill when

- scaffolding the project
- creating typed content structures
- building static-first pages
- preparing Vercel deployment

## Objective

Build a maintainable, static-first portfolio website in Next.js.

## Rules

- App Router preferred
- Tailwind CSS required
- local typed content required
- avoid server dependencies
- optimize for Vercel deployment
- keep components modular
