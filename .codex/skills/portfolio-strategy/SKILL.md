# SKILL: portfolio-strategy

## Use this skill when

- selecting featured projects
- rewriting portfolio copy
- sharpening hiring narrative
- reducing noisy career details

## Objective

Turn a long engineering history into a sharp, high-signal portfolio narrative.

## Rules

- Prefer signal over completeness
- Prefer impact over stack listing
- Prefer outcomes over adjectives
- Keep only 3 featured projects
- Rewrite vague claims into specific evidence-oriented lines

## Outputs

- hero copy
- summary copy
- featured project shortlist
- capability map bullets
